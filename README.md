## Resumen

Este curso de aproximadamente 1 hora introduce a estudiantes de secundaria a la creación de sitios web sencillos con HTML y CSS. **No requiere descargas ni instalaciones.** Solo abre tu navegador web.

**Duración total**: ~60 minutos<br>
**Público objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación<br>
**Prerrequisitos**: Ninguno<br>
**Herramientas necesarias**: Solo tu navegador web (Chrome, Firefox, Safari, Edge)

---

## Público Objetivo

- **Edad**: Estudiantes de 12 a 17 años.
- **Experiencia**: Sin experiencia previa en programación o desarrollo web.
- **Prerrequisitos**: Solo necesitan saber usar un navegador web moderno.
- **Estilo de aprendizaje**: Explicaciones cortas, ejemplos visuales, práctica inmediata en editores online y proyectos pequeños con resultados visibles.

### Lección 1: Qué es un sitio web (5 minutos)

- **Descripción**: Qué es internet, cómo funciona un sitio web y para qué sirven HTML y CSS.
- **Ver video**: [YouTube](https://www.youtube.com/watch?v=S4DXLZlkeTU&list=PLKPPKLNzMUY-RJpTt9pgY21eU9I4bAse8&index=1)
- **Conceptos clave**: Sitios web = Contenido (HTML) + Estilo (CSS).
- **Práctica**: [Abrir en CodePen](https://codepen.io/JeanArias302511/pen/bNByaQN?editors=1000) · [Código inicial](starter-files/lesson-1-web-starter.html) · [Solución completa](completed-examples/lesson-1-web-complete.html)

<br>

### Lección 2: Estructura HTML básica (10 minutos)

- **Descripción**: Aprende las etiquetas fundamentales y crea tu primera página HTML.
- **Ver video**: [YouTube](https://www.youtube.com/watch?v=IFrZGJuoBRE&list=PLKPPKLNzMUY-RJpTt9pgY21eU9I4bAse8&index=2)
- **Conceptos clave**: `<html>`, `<head>`, `<body>`, `<h1>`, `<p>`, atributos.
- **Práctica**: [Abrir en CodePen](https://codepen.io/JeanArias302511/pen/MYbdrLp?editors=1000) · [Código inicial](starter-files/lesson-2-html-starter.html) · [Solución completa](completed-examples/lesson-2-html-complete.html)

<br>

### Lección 3: Elementos HTML comunes (10 minutos)

- **Descripción**: Agrega listas, imágenes y enlaces a tu página.
- **Ver video**: [YouTube](https://www.youtube.com/watch?v=1LdO_lwPUA4&list=PLKPPKLNzMUY-RJpTt9pgY21eU9I4bAse8&index=3)
- **Conceptos clave**: `<ul>`, `<ol>`, `<li>`, `<img>`, `<a>`.
- **Práctica**: [Abrir en CodePen](https://codepen.io/Santiago-Enrique-Vargas-Alarcon/pen/WbRxYWg) · [Código inicial](starter-files/lesson-3-html-fundamentals-starter.html) · [Ejemplo completo](completed-examples/lesson-3-html-fundamentals.html)

<br>

### Lección 4: Introducción a CSS (10 minutos)

- **Descripción**: Dale color y estilo a tu HTML con CSS.
- **Ver video**: [YouTube](https://www.youtube.com/watch?v=JNMUOQSiXDY&list=PLKPPKLNzMUY-RJpTt9pgY21eU9I4bAse8&index=4)
- **Conceptos clave**: Vincular CSS, selectores, `color`, `background-color`, `font-family`.
- **Práctica**: [Abrir en CodePen](https://codepen.io/Leonardo-Delgado-the-looper/pen/MYJjgWj) · [Código inicial](starter-files/lesson-4-starter.html) · [Solución completa](completed-examples/lesson-4-complete.html)

<br>

### Lección 5: Estilo intermedio con CSS (8 minutos)

- **Descripción**: Centra contenido, agrega bordes, márgenes y relleno.
- **Ver video**: [YouTube](https://www.youtube.com/watch?v=aRNtSJqWAY4&list=PLKPPKLNzMUY-RJpTt9pgY21eU9I4bAse8&index=5)
- **Conceptos clave**: `margin`, `padding`, `border`, `text-align: center`.
- **Práctica**: [Abrir en CodePen](https://codepen.io/mauricio_pajes/pen/rajyXgR) · [Código inicial](starter-files/lesson-5-css-starter.html) · [Solución completa](completed-examples/lesson-5-css-complete.html)

<br>

### Lección 6: Proyecto - Página de perfil (12 minutos)

- **Descripción**: Combina todo lo aprendido para construir una página de perfil personal.
- **Ver video**: [YouTube](https://www.youtube.com/watch?v=zN4CG2jeoQk&list=PLKPPKLNzMUY-RJpTt9pgY21eU9I4bAse8&index=6)
- **Conceptos clave**: Integrar HTML + CSS, clases, imágenes, listas, enlaces y depuración básica.
- **Práctica**: [Crear tu perfil](https://codepen.io/mauricio_pajes/pen/VYPpJMG) · [Código inicial](starter-files/lesson-6-profile-starter.html) · [Solución completa](completed-examples/lesson-6-profile-complete.html)

<br>

### Lección 7: Errores comunes y próximos pasos (5 minutos)

- **Descripción**: Repaso de errores frecuentes en principiantes y recursos para seguir aprendiendo.
- **Ver video**: [YouTube](https://www.youtube.com/watch?v=VIylmZ-wAR4&list=PLKPPKLNzMUY-RJpTt9pgY21eU9I4bAse8&index=7)
- **Conceptos clave**: Validar código, corregir etiquetas, revisar clases CSS, completar enlaces y explorar MDN Web Docs.
- **Práctica**: [Revisar y corregir errores](https://codepen.io/mauricio_pajes/pen/KwaWOGR) · [Código inicial](starter-files/lesson-7-debug-starter.html) · [Solución completa](completed-examples/lesson-7-debug-complete.html)

## Orden recomendado

1. Leer [`course-plan.md`](course-plan.md).
2. Ver el video correspondiente a cada lección.
3. Abrir la práctica online indicada en CodePen.
4. Revisar o copiar el código de `starter-files/` para iniciar la práctica.
5. Comparar el resultado con `completed-examples/`.
6. Usar los guiones de `scripts/` como apoyo para repasar la explicación.

> [!NOTE]
> Los archivos de `starter-files/` y `completed-examples/` están pensados como material de apoyo para copiar, comparar y revisar. En CodePen, el HTML normalmente se coloca en el panel HTML y el CSS en el panel CSS.
